# PNG切分工具

一个基于React的网页工具，用于将带Alpha通道的PNG图片切分成多个矩形图片。特别适用于处理UI素材组图。

## 功能特点

- 📤 **图片上传**：支持上传 PNG
- 🎨 **可选色键抠图**：不透明背景可先取背景色抠透明，再切分；已有透明底的图可跳过抠图
- 🖌 **保护画笔**：在抠图预览上涂抹，恢复与背景同色被误抠的主体区域
- ⚙️ **高级参数**：颜色容差、羽化、边缘平滑、去溢色等（带分组说明）
- 🔍 **Alpha 通道检测**：按连通区域识别非透明像素
- ✂️ **智能切分**：每个连通区域导出为最小外接矩形 PNG
- 👀 **预览**：抠图结果 / 透明度蒙版；多语言界面
- 💾 **批量下载**：单张或全部下载切分结果

## 技术栈

- **React 18** - 前端框架
- **Vite** - 构建工具
- **Canvas API** - 图片处理

## 使用方法

1. 点击「选择 PNG 图片」上传
2. 若需去底：勾选「启用背景色抠图」，在左侧画面点击背景取色，右侧可微调参数；可用「保护画笔」修补误抠
3. 点击「开始切分」生成连通区域切分结果
4. 在结果区预览并单张或批量下载

Windows 下可在项目根目录双击 `一键启动.bat` 启动本地开发服务（需已安装 [Node.js](https://nodejs.org/)）。

## 工作原理

1. **像素检测**：读取图片的每个像素，检测Alpha通道值
2. **连通区域检测**：使用深度优先搜索（DFS）算法找到所有连通的非透明区域
3. **矩形提取**：为每个连通区域计算最小外接矩形
4. **图片切分**：使用Canvas API将每个矩形区域切分成独立的图片

## 注意事项

- 目前只支持PNG格式的图片
- 图片中的每个独立非透明区域会被切分成一个矩形
- 切分后的图片会保留原始的Alpha通道信息

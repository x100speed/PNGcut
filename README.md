# PNG切分工具

一个基于React的网页工具，用于将带Alpha通道的PNG图片切分成多个矩形图片。特别适用于处理UI素材组图。

## 功能特点

- 📤 **图片上传**：支持上传PNG格式的图片
- 🔍 **Alpha通道检测**：自动检测图片中的非透明区域
- ✂️ **智能切分**：将非透明区域切分成独立的矩形图片
- 👀 **实时预览**：预览原始图片和切分后的所有图片
- 💾 **批量下载**：支持单个下载或批量下载所有切分后的图片

## 技术栈

- **React 18** - 前端框架
- **Vite** - 构建工具
- **Canvas API** - 图片处理

```bash

```

## 使用方法

1. 点击"选择PNG图片"按钮，上传一张带Alpha通道的PNG图片
2. 系统会自动检测图片中的非透明区域
3. 将每个非透明区域切分成独立的矩形图片
4. 在结果区域查看所有切分后的图片
5. 点击单个图片的"下载"按钮下载该图片，或点击"下载全部"批量下载

## 工作原理

1. **像素检测**：读取图片的每个像素，检测Alpha通道值
2. **连通区域检测**：使用深度优先搜索（DFS）算法找到所有连通的非透明区域
3. **矩形提取**：为每个连通区域计算最小外接矩形
4. **图片切分**：使用Canvas API将每个矩形区域切分成独立的图片

## 注意事项

- 目前只支持PNG格式的图片
- 图片中的每个独立非透明区域会被切分成一个矩形
- 切分后的图片会保留原始的Alpha通道信息
